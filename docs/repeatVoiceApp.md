# オウム返しアプリ
第1回木島研ハッカソンで作成したオウム返しアプリ（音声を入力すると違う声で言葉を返してくれる）。

PCのマイクに向かってしゃべるとVOICEVOXのずんだもんの声で言葉をオウム返ししてくれる。

製作期間：2日

開発言語：Python

音声認識：Google Speech Recognition

音声合成：VOICEVOX

[GitHubリポジトリ](https://github.com/sny0/repeatVoiceApp/tree/main)
